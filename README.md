# ISS Tracker

Real-time tracking of the International Space Station using Python and Turtle graphics.

## Requirements

- Python 3.x
- geopy

## Installation

```bash
pip install geopy
```

## Usage

```bash
python sai.py
```

## Features

- Real-time ISS position tracking
- Speed calculation in km/hr
- Location identification
- Visual map display with ISS icon
- List of current astronauts on ISS
- Updates every 5 seconds

## Files

- `sai.py` - Main application
- `map.gif` - World map background
- `iss.gif` - ISS icon
